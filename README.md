# Identicon

Shell program to create profile images 250x250 like github

## Installation

If [available in Hex](https://hex.pm/docs/publish), the package can be installed
by adding `identicon` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:identicon, "~> 0.1.0"}
  ]
end
```

## Running

Clone the repository, then run `iex -S mix`, inside mix run `Identicon.main("word")` and the image is created on root folder

and published on [HexDocs](https://hexdocs.pm). Once published, the docs can
